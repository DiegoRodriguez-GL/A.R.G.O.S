# argos-reporter

HTML (Jinja2) and JSONL reporters for ARGOS. The full report template, the
compliance matrix, and the client-side filter UI land in Module 6. Module 0
ships only the base layout (`templates/base.html.j2`) wired to the design
system via `_tokens.css.j2`.
